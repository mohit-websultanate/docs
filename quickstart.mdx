- **Public API** – Used to fetch categories, games, and generate launch URLs
- **Game Launcher** – Used to start the game session via encrypted URL

#### Flow Overview

1. Operator calls APIs (categories, games)
2. Operator requests `/game-url`
3. Server returns encrypted launcher URL
4. Player is redirected to the game

#### Key Points

- All API requests are RSA signed
- Launcher URL contains encrypted session data
- No direct authentication inside the game
